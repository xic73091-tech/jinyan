# 诚言编程语言 — 完整设计文档 / Chengyan Language — Complete Design Document

## 一、语言概览 / 1. Language Overview

**诚言**（ChéngYán）是一种全中文编程语言，编译器和虚拟机完全从零构建。

**Chengyan** (ChéngYán) is a fully Chinese programming language with a compiler and virtual machine built entirely from scratch.

| 属性 | Attribute | 详情 | Details |
|------|-----------|------|---------|
| 宿主语言 | Host language | Rust（仅用于构建编译器本身，Rust 关键字无法中文化） | Rust (only used to build the compiler itself; Rust keywords cannot be Chinese-ified) |
| 执行模型 | Execution model | 源码 → 词法分析 → 语法分析 → AST → 字节码 → 栈式虚拟机解释执行 | Source → Lexing → Parsing → AST → Bytecode → Stack VM interpretation |
| 类型系统 | Type system | 动态类型（可选静态类型推断） | Dynamic typing (optional static type inference) |
| 范式 | Paradigm | 过程式为主，支持面向对象和函数式特性 | Primarily procedural, with OOP and functional features |
| 文件扩展名 | File extensions | `.jy`（源码）、`.jbc`（字节码） | `.jy` (source), `.jbc` (bytecode) |

## 四、编译器架构 / 4. Compiler Architecture

```
源代码 (.jy) / Source (.jy)
    │
    ▼
┌─────────────────────┐
│     词法分析器       │  源码字符流 → 记号流
│     Lexer           │  Source char stream → Token stream
│                     │  支持 UTF-8 中文 / Supports UTF-8 Chinese
└─────────┬───────────┘
          │ Vec<记号> / Vec<Token>
          ▼
┌─────────────────────┐
│     语法分析器       │  记号流 → 抽象语法树
│     Parser          │  Token stream → AST
│                     │  递归下降法 / Recursive descent
└─────────┬───────────┘
          │ 程序 (AST) / Program (AST)
          ▼
┌─────────────────────┐
│     字节码编译器     │  AST → 字节码指令序列
│     Compiler        │  AST → Bytecode instruction sequence
│                     │  基于栈的指令集 / Stack-based ISA
└─────────┬───────────┘
          │ 字节码块 (指令 + 常量池) / Bytecode chunk (instructions + constant pool)
          ▼
┌─────────────────────────────────────────┐
│           虚拟机 / VM                    │
│  ┌──────────┐  ┌──────────┐            │
│  │ 指令分发  │  │ 操作数栈 │            │
│  │ Dispatch  │  │ Operand  │            │
│  │ loop     │  │ stack    │            │
│  └──────────┘  └──────────┘            │
│  ┌──────────┐  ┌──────────┐            │
│  │ 调用栈   │  │ 全局变量 │            │
│  │ Call     │  │ Global   │            │
│  │ stack    │  │ vars     │            │
│  └──────────┘  └──────────┘            │
│  ┌──────────────────────────────────┐  │
│  │   垃圾回收器 / Garbage Collector  │  │
│  │   标记-清除算法 / Mark-Sweep      │  │
│  └──────────────────────────────────┘  │
│  ┌──────────────────────────────────┐  │
│  │   内置函数 / Built-in Functions   │  │
│  │   打印、输入、文本化、长度 等     │  │
│  │   print, input, to-string, etc.  │  │
│  └──────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

## 六、示例执行流程 / 6. Example Execution Flow

### 源代码 / Source Code (`示例.jy`)
```诚言
定义 主函数() {
    让 消息 = "你好，世界！"
    打印(消息)
}
主函数()
```

### 词法分析输出 / Lexer Output
```
[定义] [标识符:"主函数"] [左圆括号] [右圆括号] [左花括号]
[让] [标识符:"消息"] [赋值] [文本字面量:"你好，世界！"]
[标识符:"打印"] [左圆括号] [标识符:"消息"] [右圆括号]
[右花括号]
[标识符:"主函数"] [左圆括号] [右圆括号]
[文件结束]
```

| 记号 | Token | 含义 | Meaning |
|------|-------|------|---------|
| `定义` | `define` | 函数定义关键字 | Function definition keyword |
| `标识符:"主函数"` | `Identifier:"main"` | 函数名 | Function name |
| `左圆括号` / `右圆括号` | `LParen` / `RParen` | 参数列表 | Parameter list |
| `左花括号` / `右花括号` | `LBrace` / `RBrace` | 函数体 | Function body |
| `让` | `let` | 变量声明关键字 | Variable declaration keyword |
| `赋值` | `Assign` | 赋值运算符 | Assignment operator |
| `文本字面量` | `StringLiteral` | 字符串值 | String value |
| `文件结束` | `EOF` | 输入结束 | End of input |

### 生成的字节码 / Generated Bytecode
```
=== 常量池 / Constant Pool ===
[0] 文本("你好，世界！") / String("Hello, World!")
[1] 函数("主函数", 参数:0, 局部:1, 地址:0) / Function("main", args:0, locals:1, addr:0)

=== 函数 主函数 / Function main (地址/addr 0-9) ===
  0: 压入常量 0      // "你好，世界！" / "Hello, World!"
  2: 设置局部 0      // 消息 / message
  4: 读取全局 "打印"  // "print"
  6: 读取局部 0      // 消息 / message
  8: 调用 1          // call 1
  9: 弹出            // pop
 10: 压入空          // push null
 11: 返回            // return

=== 入口 / Entry Point ===
 12: 读取全局 "主函数" // "main"
 14: 调用 0           // call 0
 15: 弹出             // pop
 16: 停止             // stop
```

### 虚拟机执行输出 / VM Output
```
你好，世界！
Hello, World!
```
